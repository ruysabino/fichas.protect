⚙️ Definições

💾 Backup — Exportar dados

Exporta todas as fichas e clientes para um ficheiro cifrado (.brb). Guarde-o num local seguro (Google Drive, Dropbox, pen USB).

📂 Restaurar dados

Importa um ficheiro de backup .brb anteriormente exportado.

📋 Alternativa para iPad / iPhone

Se o botão "Selecionar ficheiro" não funcionar, abra o ficheiro .brb em qualquer app de texto (Ficheiros, Notas, etc.), copie todo o conteúdo e cole aqui:

📋 Relatório RGPD

Gera um relatório com todos os dados pessoais armazenados, em formato JSON e HTML legível. Útil para auditorias e cumprimento do Regulamento Geral de Proteção de Dados.

⚠️ Zona de Perigo

O reset apaga permanentemente todos os dados da aplicação: fichas, clientes e utilizadores. Esta acção não pode ser desfeita. Efectue um backup antes de continuar.
