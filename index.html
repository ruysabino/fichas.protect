⚙️ Definições

💾 Backup — Exportar dados

Exporta todas as fichas e clientes para um ficheiro cifrado (.brb). Guarde-o num local seguro (Google Drive, Dropbox, pen USB).

📂 Restaurar dados

Importa um ficheiro de backup .brb anteriormente exportado.

📋 Relatório RGPD

Gera um relatório com todos os dados pessoais armazenados, em formato JSON e HTML legível. Útil para auditorias e cumprimento do Regulamento Geral de Proteção de Dados.
